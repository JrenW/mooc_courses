Browser Sync is your Web Dev pal!

What's Browser Sync?

Browser Sync does automatic real-time synchronization of your edits in your local script, be it html, css, or javascript, and updates webpage view once you 'save' the edited script.

So you don't have to re-run entire script on browser every time!

How to get it working?

Once you have successfully installed Browser Sync, do the following:

  1. Navigate to your target directory in which you want Browser Sync to WATCH (i.e. to keep track and auto-sync);

  2. Open up your terminal and type this command line (Mac only):

    [ browser-sync start --server --directory --files "*" ]

  3. A Browser Sync webpage should pop up for you to visualize all folders/files in the target directory;

  4. Click on the file on the Broswer Sync page you would like to sync-view;

  5. Now you can edit scripts locally and view updates by SAVE your script and REFRESH the Broswer Sync page;

  6. Once you have finished editing, EXIT Broswer Sync by hitting "Ctrl + C" in the command line (Mac only);
