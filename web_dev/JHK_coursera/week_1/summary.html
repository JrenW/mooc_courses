Module 1: HTML Basics

What does HTML stand for?

= HyperText Markup Language
  • Hypertext: texts contain links
  • Markup: to annotate using "tags"
  • Language: syntax matter; browser reads it

What does HTML do?

  • Define document structures
  • Annotation hypertext content

What are the three pillars of web technologies?

  • HTML: structure design
  • CSS: visual styling
  • Javascript: flow of control

What are the two organizations making HTML standards?

  • W3C: World Wide Web Consortium
  • WHATWG: Web Hypertext Application Technology Working Group
*W3C updates versions of HTML while WHATWG does not; W3C also provides a syntax validator

What's the anatomy of an HTML tag?

OPENING TAG + contents + CLOSING TAG
e.g. < p > Hello World < /p >

Key features of HTML tags:

  • All HTML5 tags must have an opening tag
  • Some don't have closing tags, e.g. < br > which adds a blank line on your webpage
  • A tag has attributes, e.g. < p id="haha"> , where id is the attribute name AND "haha is the attribute value.
  • HTML doesn't differentiate single vs double quotation: id="haha" or id='haha'
  • Order of closing matters! = the opposite order of opening tags

What's a page declaration?

< !DOCTYPE html >
*The declaration is NOT an HTML tag; it tells the web browser about what version of HTML the page is written in (e.g. HTML5).

What's the Quirks mode?

If you missed the page declaration, the browser will interpret your HTML in Quirks mode as the browser assume that this webpage will not follow the HTML standard, e.g. styles won't work correctly etc.

What's a minimal version of HTML script?

< html > - < meta > - < head > - < title > - < body >

What's a meta tag?

Meta tags communication Webpage info to the browser, not the server. A meta tag is between < head > and < title > tells search engine core features of your webpage, e.g. < meta charset="utf-8">. A well-written meta tage can attract more click-throughs from a search engine result!

How does a browser interpret an HTML script?

A browser will ALWAYS interpret HTML sequentially, top to bottom
